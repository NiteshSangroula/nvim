My Nvim setup:

1. Install neovim v0.5 
2. Clone nvim configuration file from github nvim repository.
3. Install vim-plug.
4. Install Python, python3, nodejs, npm.
5. Run command "pip3 install neovim"
6. Run "PlugInstall" on neovim
7. Finish with required coc extensions
Note:
You must install following font:
1. CaskaydiaCove Nerd font
or
2. DejaVuSansMono NF

in order for icons to work properly
